# simple-llm-app-fastapi-langchain

## Introduction
The simple-llm-app-fastapi-langchain project is designed to provide a basic structure for building language model applications using FastAPI and LangChain.

## Endpoints
The following endpoints are available:
- **GET /health**: Returns a health check response.
- **GET /healthsecond**: Returns a secondary health check response.
- **GET /healththird**: Returns a tertiary health check response.
- **POST /ingest**: Ingests documents into FAISS.

## Usage
To use the application, start by running the FastAPI server. You can then access the available endpoints using a tool like curl or a web browser.

## Development
To contribute to the project, clone the repository and install the required dependencies. You can then run the application in development mode to test your changes.
